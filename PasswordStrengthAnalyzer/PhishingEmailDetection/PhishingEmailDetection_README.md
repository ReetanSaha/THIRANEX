# Phishing Email Detection Model

## Overview

This project uses Machine Learning to classify emails as phishing or safe.

## Features

- Detect phishing emails
- Text preprocessing
- TF-IDF feature extraction
- Naive Bayes classifier
- Confusion matrix visualization
- Save trained model

## Technologies Used

- Python
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn

## Folder Structure

dataset/
models/
reports/

## How to Run

1. Clone repository.
2. Install dependencies.

pip install -r requirements.txt

3. Run:

# Run this single command from anywhere:
~/Documents/PhishingEmailDetection/venv/bin/python ~/Documents/PhishingEmailDetection/phishing_detector.py

# Similarly, for prediction:
~/Documents/PhishingEmailDetection/venv/bin/python ~/Documents/PhishingEmailDetection/predict_email.py


## Output

- Accuracy score
- Confusion matrix
- Email classification
